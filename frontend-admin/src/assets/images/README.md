# Images Assets

## Background Image pour Login

Placez votre image de campus ici avec le nom : `campus-background.jpg`

L'image fournie dans le screenshot doit être sauvegardée comme :
- **Nom** : `campus-background.jpg`
- **Format** : JPG ou PNG
- **Résolution recommandée** : 1920x1080 ou plus
- **Taille** : < 500KB (optimisée)

## Alternative

Si l'image n'est pas disponible, un dégradé violet sera utilisé comme fallback.

Pour ajouter l'image :
1. Sauvegardez l'image du campus comme `campus-background.jpg`
2. Placez-la dans ce dossier (`src/assets/images/`)
3. L'application l'utilisera automatiquement
