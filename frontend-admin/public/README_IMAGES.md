# Images requises pour l'application

Pour que l'application fonctionne correctement, vous devez ajouter les images suivantes dans le dossier `public/` :

## 1. Logo principal (logo1.png)
- **Emplacement** : `public/logo1.png`
- **Description** : Logo EQUIZZ utilisé dans la page de login et la page d'accueil
- **Dimensions recommandées** : 200x80px (ou ratio similaire)
- **Format** : PNG avec fond transparent

## 2. Illustration principale (Equizz3.jpg)
- **Emplacement** : `public/Equizz3.jpg`
- **Description** : Illustration utilisée dans la page de login et la page d'accueil
- **Dimensions recommandées** : 800x600px minimum
- **Format** : JPG ou PNG

## Note
En attendant d'ajouter vos propres images, vous pouvez :
1. Créer des images placeholder
2. Utiliser des images de stock
3. Ou simplement laisser les chemins vides (l'application affichera un placeholder)
